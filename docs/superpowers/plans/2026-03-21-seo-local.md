# SEO Local — Enza Duhem Musicothérapeute — Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Optimiser le référencement local du site musicotherapeute-hainaut.be pour apparaître dans les premiers résultats Google pour "musicothérapie Tournai", "musicothérapeute Hainaut" et recherches locales connexes.

**Architecture:** Modifications purement statiques — balises HTML dans `<head>` de chaque page, bloc JSON-LD en `@graph`, Open Graph, canonicals, et activation de la redirection www dans `netlify.toml`. Aucune dépendance externe ajoutée. Aucun framework de test disponible — validation via outils en ligne (Rich Results Test, W3C).

**Tech Stack:** HTML statique, JSON-LD (schema.org), Netlify (hébergement + redirects), GitHub (déploiement auto)

**Spec:** `docs/superpowers/specs/2026-03-21-seo-local-design.md`
